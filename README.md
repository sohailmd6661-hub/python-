<h1>Basic Python</h1>

<h2>Introduction</h2>
<p>This document covers basic Python concepts with examples.</p>

<h2>Variables</h2>
<p>Variables are used to store data values.</p>
<pre>
x = 10
name = "Sohail"
print(x, name)
</pre>

<h2>Data Types</h2>
<ul>
  <li>int</li>
  <li>float</li>
  <li>string</li>
  <li>list</li>
  <li>tuple</li>
  <li>set</li>
  <li>dictionary</li>
</ul>

<h2>String Example</h2>
<pre>
s = "Python"
print(s.upper())
print(len(s))
</pre>

<h2>List Example</h2>
<pre>
numbers = [1, 2, 3, 4, 5]
numbers.append(6)
print(numbers)
</pre>

<h2>For Loop</h2>
<pre>
for i in range(1, 6):
    print(i)
</pre>

<h2>While Loop</h2>
<pre>
i = 1
while i <= 5:
    print(i)
    i += 1
</pre>

<h2>Function Example</h2>
<pre>
def add(a, b):
    return a + b

result = add(10, 20)
print(result)
</pre>
